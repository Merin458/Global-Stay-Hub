# 🌍 Global Stay Hub

Global Stay Hub is a web-based accommodation platform designed for international students to find safe and affordable housing. It connects students, house owners, and administrators in a single system, making the process of listing, booking, and managing accommodations simple and efficient.

## 🚀 Features

### 👤 Student Module
- Register and login
- Search and view available accommodations
- Book rooms easily
- View booking status

### 🏠 House Owner Module
- Add and manage property listings
- Update availability and details
- View booking requests

### 🛠️ Admin Module
- Manage users (students & owners)
- Monitor listings and bookings
- Maintain overall system control

## 💻 Tech Stack

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL

## ⚙️ Installation

1. Clone the repository:
git clone https://github.com/your-username/global-stay-hub.git

2. Move the project to your server folder (e.g., XAMPP htdocs)

3. Import the database:
- Open phpMyAdmin
- Create a database
- Import the .sql file

4. Configure database connection in config.php

5. Run the project:
http://localhost/global-stay-hub
